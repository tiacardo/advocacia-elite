## ⚙️ Como Realizar Manutenções (Via Celular)

Para atualizar o site diretamente pelo seu smartphone:

1.  **Atualizar Preço/Link**: Localize a função `checkout()` no final do arquivo `index.html` e altere a URL da Kiwify.
2.  **Alterar Textos**: Procure pelas tags `<h1>`, `<h2>` ou `<p>` para ajustar a copy de vendas.
3.  **Deploy Automático**: Qualquer alteração salva ("Commit") no GitHub será refletida automaticamente no link da Vercel em segundos.
